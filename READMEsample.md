## Description 
Bootcamp week 11 challenge usind node.js and npm inquirer v6.5.0 to create a computer generated readme file based on prompted user input.

## Installation 
Git clone or copy the repository into your computer and type "node index.js" in yout terminal to run the programme. 

## Usage 
This ReadMe generator creates a Table of contents based on the sections populated with user provided information. If no information is given, the section does not display in the file and the table of contents does not include it. A license badge is shown under the Title section when the user selects the project license. A link to GitHub pages and email appears in the Questions section.
